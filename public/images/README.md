# Product Images

Загрузите сюда фотографии товаров для каталога.

## Требования к изображениям

### Формат и размер
- **Формат**: JPG, PNG или WebP
- **Соотношение сторон**: 3:4 (портретная ориентация)
- **Минимальное разрешение**: 600x800 пикселей
- **Рекомендуемое разрешение**: 900x1200 пикселей
- **Размер файла**: до 500 КБ (оптимизируйте для web)

### Naming Convention (Именование файлов)
Назовите файлы по следующему шаблону:
```
product1.jpg
product2.jpg
product3.jpg
product4.jpg
product5.jpg
product6.jpg
product7.jpg
product8.jpg
...
```

### Рекомендации по фото
- Светлый или белый фон
- Хорошее освещение
- Модель в центре кадра
- Вертикальная ориентация
- Профессиональное качество

## Как добавить фото

1. Переименуйте ваши фотографии согласно шаблону выше
2. Скопируйте их в эту папку (`public/images/`)
3. Фотографии автоматически отобразятся в каталоге

## Обновление данных о товарах

Для изменения названий и цен товаров отредактируйте файл:
`src/components/ProductGrid.jsx`

В массиве `products` измените:
- `name` - название товара
- `price` - цена
- `priceFrom` - цена "от" (для товаров с вариациями)
- `image` - путь к изображению (должен соответствовать имени файла)
